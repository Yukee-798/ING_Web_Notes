         还能将字符串中符合规则的子串提取出来

二、创建正则表达式对象

    2. test()
        作用：用于检测一个字符串是否符合该正则表达式的规则，返回true/false

 ##  正则表达式的创建
---
一个完整的正则表达式包含 `pattern`(模式) 和 `modifiers`(修饰符)，形如：`/模式/修饰符`

```javascript
var reg = /abc/g; // 表示在全局去匹配 'abc' 这个子串
```

## 修饰符
---
* `i` 表示忽略大小写的匹配模式
* `g` 表示全局匹配模式

## 基本用法
---
```javascript
// 全局匹配 'a' 或 'b' 或 'c' 这几个字符
var reg = /a|b|c/g;
var reg = /[abc]/;
reg.test('defa'); // 返回 true，因为 'defa' 符合该正则表达式

// 忽略大小写并全局匹配 'aBc' 或 'def' 这两个字符串
var reg = /aBc|def/ig;

// 转义字符的使用
var reg1 = /\\/g; // 判断是否有 '\'
var reg2 = /\+/g; // 判断是否有 '+'
var reg3 = /\*/g; // 判断是否有 '*'
```

## 方括号的使用
---
```javascript
// 忽略大小写全局匹配方括号内的字符
var reg = /[123abc]/ig; // 只要原字符串包含其中的某个字符就会匹配上

// 检查原字符串中是否包含数字，匹配所有的数字字符
var reg = /[0-9]/g;

// 检查原字符串中是否包含字母，匹配所有的字母字符
var reg = /[A-z]/g; // 忽略大小写
var reg = /[a-z]/ig;

// 检查原字符串中是否包含字母或数字，匹配所有的数字和字母字符
var reg = /[A-z0-9]/g;

// 检查原字符串中是否包含中文字符，匹配所有的中文字符
var reg = /[\u4e00-\u9fa5]/g; // 4e00 到 9fa5 是中文的 Unicode 码

// 检查原字符串中是否不包含方括号内的字符，匹配除了放括号内的字符外所有字符
var reg = /[^abc]/g;
```

## 量词的使用
---

```javascript
// 是否包含连续出现3次的a字符串，匹配字符串中 'aaa' 子串
var reg = /a{3}/g;

// 是否包含连续出现 1～3 次的 'abc' 字符串
var reg = /(abc){1,3}/g;

// 是否包含连续3个以上的字母字符，匹配字符串至少包含3个
var reg = /[A-z]{3,}/g;

// n+ 表示 n{1,}
// n* 表示 n{0,}
// n? 表示 n{0,1}

// 是否以'123'开头
var reg = /^(123)/g;

// 是否以'123'结尾
var reg = /(123)$/g;

// 检测一个字符串是否为合法手机号
// 首位为1，第二位为3-9，后9位为0-9
var reg = /^1[3-9][0-9]{9}$/g;

// 判断字符串是否为中文字符或英语字母的正则表达式
var reg = /^([\u4e00-\u9fa5]|[A-z]){1,}$/g;
var reg = /^([\u4e00-\u9fa5]|[A-z])+$/g;

// 判断字符串是否为数字
var reg = /^[0-9]{1,}$/g;
var reg = /^[0-9]+/g;

// 匹配所有'is'后面接着 ' all'的子串
var str = "Is this all there is";
var arr = str.match(/is(?= all)/g);

// 匹配所有'is'后面没有接着 ' all'的子串
var str = "Is this all there is";
var arr = str.match(/is(?! all)/ig);
```

## 元字符的使用
---
```javascript
/*

任意字符占位
	. 可以表示任意字符
	
数字字母下划线占位
	\w 可以表示数字、字母、下划线，等价于 [A-z0-9_]
	\W 非数字、字母、下划线，等价于 [^A-z0-9_]
	
数字占位
	\d 表示数字，等价于 [0-9]
	\D 非数字，等价于 [^0-9]

空白字符占位
	\s 表示空白字符
	\S 非空白字符

单词边界占位
	\b 表示单词边界
	\B 非单词边界

特殊符合占位
	\n 表示换行符
	\t 表示制表符
	\r 表示回车

注意：这里的占位表示在正则表达式中的占位

*/

// 删除字符串中所有字母
var str = '909abcd397sdad116';
str = str.replace(/[a-z]/gi, '');

// 匹配句子中的 child 单词
var str = 'children child children child'
var arr = str.match(/\bchild\b/g);

// 去除字符串开头和末尾的空格
var reg = /^\s*|\s*$/g;
var str = '        ab  c         ';
str = str.replace(reg, '');

// 以 换行符、制表符、回车、逗号、空格、减号、/ 、* 、\ 、下划线 来分割字符串
var reg = /[\n\t\r,\s-/\*\\_]+/g;
var str = 'abc,def xyz---omg/bili\\didi/momo*123';
var strArr = str.split(reg);

/*
[
  'abc',     'def',
  'xyz-omg', 'bili',
  'didi',    'momo',
  '123'
]
*/

// 判断输入的字符串是否是标准的邮箱格式
// 规则：
// 		1. 以长度至少为3的任意字母、数字、下划线开头
// 		2. 后面可能紧接着一个 . 然后长度至少为1的任意数字、字母、下划线
//		3. 接着有一个 @
// 		4. 接着就是长度至少为1的任意数字、字母
//		5. 最后以 . 和 长度为{2,5}的字母结尾
var reg = /^\w{3,}(\.\w+)*@{0-9a-z}+(\.[a-z]{2,5})$/ig;
reg.test('90921399@126.moc'); // true
reg.test('k909392221.momo@qq.com'); // true
```

## 小结
---
   1. 当一个子串匹配完后，该子串相当于删除，接着往后面匹配
   2. 有量词的，是从最大量开始匹配
   3. 带上中括号的表示匹配字符
   4. 中括号结合量词使用可以匹配任意长度符合规则的子串，且长度是根据指定的最大长度收缩着匹配
   5. 开全局g则表示匹配所有符合规则的，不开全局则只匹配第一个
   6. split 就算不开全局也相当于开了全局
   7. search 开了全局也没用，相当于不开
